# Contributors

Wholesail is built and maintained by the people listed below. We welcome
issues, discussion, and PRs — but please open an issue first for anything
larger than a typo or one-line fix.

## Maintainers

- **Deep Patel** ([@reallydeep](https://github.com/reallydeep)) — founder,
  product, and lead engineer

## How to be added

1. Open a pull request that lands a non-trivial change (feature, bug fix,
   docs improvement, or test coverage).
2. Add yourself to this file in the same PR under the appropriate section.
3. Ask a maintainer to merge.
